## 🚀 启动指南

### 1. 数据库初始化
```sql
CREATE DATABASE teaching_employment CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
```

### 2. 修改配置
编辑 `application.yml`:
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/teaching_employment
    username: root
    password: your_password

jwt:
  secret: your-secret-key
  expiration: 604800000  # 7天

file:
  upload:
    path: uploads
    domain: http://localhost:8083
```

### 3. 启动应用
```bash
mvn clean package
java -jar target/teaching-employment.jar
```

### 4. 访问API文档
打开浏览器访问: http://localhost:8083/doc.html
