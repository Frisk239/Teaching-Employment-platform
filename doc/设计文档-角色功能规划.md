## 🎯 系统角色概览

### 角色矩阵

| 角色 | 权限级别 | 核心职责 | 开发优先级 |
|------|---------|---------|-----------|
| **学员** | 基础用户 | 学习、求职、提交作业 | ⭐⭐⭐⭐⭐ |
| **教师** | 教学管理 | 批改作业、指导就业 | ⭐⭐⭐⭐ |
| **企业对接人** | 企业端 | 发布职位、筛选简历 | ⭐⭐⭐⭐ |
| **学院负责人** | 院系管理 | 数据统计、审批管理 | ⭐⭐⭐ |
| **管理员** | 系统管理 | 全局配置、用户管理 | ✅ 已完成 |

## 👨‍🎓 学员角色设计

### 角色定位
学员是系统的核心用户，所有教学和就业流程围绕学员展开。

### 核心功能模块

#### 1️⃣ 个人中心

**功能描述**: 学员个人信息管理

**核心功能**:
- 📝 查看和编辑个人基本信息
  - 姓名、学号、性别、出生日期
  - 联系方式（手机、邮箱）
  - 专业、班级、入学年份
- 📄 简历管理
  - 上传简历（支持PDF、Word格式）
  - 预览简历
  - 更新简历版本
- 🔒 密码修改
- 🎓 教育经历
  - 添加/编辑教育背景
  - 学历、专业、时间范围
- 💼 项目经历
  - 添加/编辑项目经验
  - 项目名称、角色、描述、技术栈

**数据表**: `student`, `user`, `student_profile`

---

#### 2️⃣ 职位管理

**功能描述**: 浏览和申请职位

**核心功能**:
- 📋 职位列表
  - 查看所有已发布职位
  - 职位详情（薪资、要求、福利、公司信息）
  - 搜索和筛选
    - 按职位名称搜索
    - 按公司筛选
    - 按薪资范围筛选
    - 按工作地点筛选
    - 按职位类型筛选（实习/全职/校招）
- 🏠 公司详情
  - 查看公司介绍
  - 查看公司所有职位
- 📊 职位统计
  - 查看投递该职位的人数
  - 查看该职位的招聘进度
- ✨ 推荐职位
  - 根据专业智能推荐
  - 根据历史投递记录推荐

**数据表**: `position`, `company`, `job_application`

---

#### 3️⃣ 投递管理

**功能描述**: 职位申请和状态追踪

**核心功能**:
- 📤 一键投递
  - 选择职位投递
  - 自动附带简历
  - 添加求职信（可选）
- 📝 我的申请
  - 查看所有申请记录
  - 申请状态筛选
    - 待处理
    - 已筛选
    - 笔试中
    - 面试中
    - 已录用
    - 已拒绝
- 📊 申请详情
  - 查看申请进度
  - 查看职位信息
  - 查看公司反馈
- ⏰ 状态变更通知
  - 简历被查看
  - 进入笔试
  - 进入面试
  - 收到Offer

**数据表**: `job_application`, `application_status_log`

**状态流转图**:
```
待处理 → 已筛选 → 笔试中 → 面试中 → 已录用/已拒绝
```

---

#### 4️⃣ 面试管理

**功能描述**: 面试安排和记录

**核心功能**:
- 📅 我的面试
  - 查看所有面试安排
  - 面试时间、地点、类型
  - 面试官信息
  - 职位信息
- 🔔 面试提醒
  - 即将到来的面试（24小时内）
  - 面试时间变更通知
- 📝 面试记录
  - 查看历史面试记录
  - 面试结果（通过/不通过）
  - 面试反馈
- 📍 面试准备
  - 查看公司地址
  - 查看职位要求
  - 查看面试建议

**数据表**: `interview`

---

#### 5️⃣ Offer管理

**功能描述**: 查看和管理收到的Offer

**核心功能**:
- 🎉 我的Offer
  - 查看所有收到的Offer
  - Offer详情
    - 薪资待遇
    - 入职时间
    - 工作地点
    - 福利待遇
  - Offer状态
    - 待接受
    - 已接受
    - 已拒绝
    - 已过期
- ✅ Offer操作
  - 接受Offer
  - 拒绝Offer
  - 查看Offer文档（PDF）
- 📊 Offer统计
  - 收到的Offer数量
  - 平均薪资
  - 薪资分布

**数据表**: `offer`

---

#### 6️⃣ 课程学习

**功能描述**: 课程管理和学习进度

**核心功能**:
- 📚 我的课程
  - 查看已选课程列表
  - 课程进度
  - 课程大纲
  - 课程资源下载
- 📖 学习进度
  - 完成的课程数
  - 进行中的课程
  - 课程成绩
- 📝 课程作业
  - 查看作业列表
  - 提交作业
  - 查看作业成绩和反馈
  - 作业截止日期提醒

**数据表**: `course`, `course_student`, `assignment`

---

#### 7️⃣ 日报管理

**功能描述**: 每日学习记录

**核心功能**:
- 📝 写日报
  - 填写日报内容
    - 今日学习内容
    - 学习进度
    - 遇到的问题
    - 明日计划
  - 上传附件（代码、截图等）
  - 记录学习时长
  - 记录代码行数
- 📊 我的日报
  - 查看历史日报
  - 日报状态
    - 草稿
    - 已提交
    - 已批阅
  - 查看教师评语和评分
- 📈 学习统计
  - 本周日报数量
  - 累计学习时长
  - 累计代码行数
  - 学习时长趋势图

**数据表**: `daily_report`

#### 8️⃣ 作业管理 🆕

**功能描述**: 课程作业提交和管理

**核心功能**:
- 📋 作业列表
  - 查看所有课程的作业
  - 作业状态筛选
    - 未开始
    - 进行中
    - 已提交
    - 已批改
    - 已逾期
  - 作业截止时间排序
- 📤 提交作业
  - 在线填写作业内容
  - 上传作业文件（支持多文件）
    - 代码文件（.zip, .rar）
    - 文档文件（.pdf, .doc, .docx）
    - 图片文件（.png, .jpg, .jpeg）
  - 添加作业说明
  - 保存草稿
  - 提交作业
- 📝 作业详情
  - 查看作业要求
  - 查看作业资源
  - 查看提交记录
  - 查看教师批改结果
  - 查看作业分数
  - 查看教师评语
- 📊 作业统计
  - 已完成作业数
  - 待完成作业数
  - 平均分数
  - 作业完成率
- ⏰ 提醒功能
  - 即将截止的作业（24小时内）
  - 逾期未提交提醒

**数据表**: `assignment`, `assignment_submission`

**作业状态流转**:
```
未开始 → 进行中 → 已提交 → 已批改
         ↓
       已逾期
```

#### 3️⃣ 课程管理

**功能描述**: 课程创建和管理

- 📝 课程公告

#### 4️⃣ 作业管理 🆕

**功能描述**: 作业发布和批改

**核心功能**:
- 📋 作业列表
  - 查看所有课程的作业
  - 按课程筛选
  - 按状态筛选
    - 未发布
    - 进行中
    - 已截止
  - 创建作业
    - 作业标题
    - 作业描述
    - 所属课程
    - 截止时间
    - 满分分数
    - 作业要求
    - 上传作业资源
- ✍️ 批改作业
  - 查看待批改作业列表
  - 按提交时间排序
  - 查看学生作业内容
  - 下载学生作业文件
  - 在线批改
    - 给出分数
    - 添加批改评语
    - 支持富文本编辑
    - 添加批注（支持图片标注）
  - 批改统计
    - 已批改数量
    - 待批改数量
    - 批改进度
- 📊 作业统计
  - 提交率统计
  - 成绩分布
  - 平均分
  - 优秀作业展示
- 📤 作业发布
  - 编辑作业
  - 发布作业
  - 重新发布
  - 删除作业

**数据表**: `assignment`, `assignment_submission`

**作业批改流程**:
```
学生提交作业 → 教师批改作业 → 给出分数和评语 → 学生查看结果
```

**批改界面功能**:
- 左侧：学生提交列表
- 右侧：作业详情和批改区域
  - 学生信息和提交时间
  - 作业内容展示
  - 文件下载
  - 分数输入（0-满分）
  - 评语编辑器（富文本）
  - 批注工具
    - 文字批注
    - 图片标注
    - 语音批注
- 提交批改按钮
- 批改历史查看

### 数据库设计

#### 1. 作业表 (assignment)

```sql
CREATE TABLE assignment (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '作业ID',
    course_id BIGINT NOT NULL COMMENT '课程ID',
    title VARCHAR(200) NOT NULL COMMENT '作业标题',
    description TEXT COMMENT '作业描述',
    requirements TEXT COMMENT '作业要求',
    max_score INT DEFAULT 100 COMMENT '满分分数',
    deadline DATETIME NOT NULL COMMENT '截止时间',
    status VARCHAR(20) DEFAULT 'draft' COMMENT '状态: draft-草稿, published-已发布, closed-已截止',
    attachment_url VARCHAR(500) COMMENT '作业附件URL',
    created_by BIGINT COMMENT '创建人ID',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    INDEX idx_course_id (course_id),
    INDEX idx_status (status),
    INDEX idx_deadline (deadline)
) COMMENT='作业表';
```

#### 2. 作业提交表 (assignment_submission)

```sql
CREATE TABLE assignment_submission (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '提交ID',
    assignment_id BIGINT NOT NULL COMMENT '作业ID',
    student_id BIGINT NOT NULL COMMENT '学生ID',
    content TEXT COMMENT '作业内容',
    attachment_urls VARCHAR(1000) COMMENT '附件URLs（多个文件用逗号分隔）',
    score DECIMAL(5,2) COMMENT '得分',
    comment TEXT COMMENT '教师评语',
    status VARCHAR(20) DEFAULT 'draft' COMMENT '状态: draft-草稿, submitted-已提交, graded-已批改',
    submit_time DATETIME COMMENT '提交时间',
    grade_time DATETIME COMMENT '批改时间',
    graded_by BIGINT COMMENT '批改人ID',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    UNIQUE KEY uk_assignment_student (assignment_id, student_id),
    INDEX idx_student_id (student_id),
    INDEX idx_status (status),
    INDEX idx_submit_time (submit_time)
) COMMENT='作业提交表';
```
